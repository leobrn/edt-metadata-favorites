# Избранное метаданных (EDT Metadata Favorites) - руководство пользователя

Плагин для 1С:EDT позволяет добавлять нужные проекты и объекты метаданных в **избранное** в дереве
Навигатора и одной кнопкой скрывать всё лишнее, оставляя на виду только избранное. Удобно, когда
в воркспейсе много конфигураций и объектов, а работаете вы каждый день с небольшим их числом.

## Что умеет плагин

- Добавлять проект (конфигурацию) целиком в избранное.
- Добавлять в избранное отдельные объекты метаданных (справочники, документы, регистры и т.п.)
  и их вложенные объекты (реквизиты, табличные части, формы и др.).
- Помечать избранное значком-звездой прямо в дереве.
- Прятать всё, что не добавлено в избранное, кнопкой-фильтром на панели Навигатора.
- Массово управлять избранным в отдельной форме с поиском и группами типов метаданных.

## Как добавлять и удалять из избранного

Все команды - в **контекстном меню** дерева Навигатора (правая кнопка мыши по элементу). Набор
пунктов зависит от того, на чём вы стоите и что уже добавлено в избранное, - показываются только команды,
которые реально что-то изменят.

### На проекте (конфигурации)

- **Добавить проект в избранное / Удалить проект из избранного** - изменяет состояние всего проекта.
- **Удалить все объекты из избранного** - удаляет из избранного все отдельные объекты внутри проекта.
  Пункт появляется, только если внутри есть избранные объекты. На состояние самого проекта
  не влияет.

### На объекте метаданных

- **Добавить в избранное / Удалить из избранного** - изменяет состояние выбранного объекта.
- **Добавить в избранное с вложенными** - добавляет всю ветку в избранное с наследованием. Существующие
  вложенные объекты добавляются сразу, а созданные позднее формы, реквизиты и другие потомки
  автоматически становятся избранными.
- **Удалить из избранного с вложенными** - удаляет из избранного объект и всю ветку. Если выше находится
  ветка, добавленная в избранное с вложенными, удалённая ветка сохраняется как исключение.

> Обычное «Добавить в избранное» добавляет только выбранный объект. Наследование включается
> исключительно командой «Добавить в избранное с вложенными». Отдельный потомок можно снять вручную —
> он сохранится как исключение.

Ветки, добавленные в избранное командой «с вложенными» в версиях плагина до 1.2.0, остаются разовыми снимками.
Чтобы включить наследование для такой ветки, повторно выполните «Добавить в избранное с вложенными».

## Форма управления избранным

Открыть форму можно сочетанием **Ctrl+Alt+F** или командой **Избранное** в дополнительном меню
Навигатора.

В форме доступны:

- переключение между открытыми проектами конфигураций;
- поиск по имени объекта;
- групповой выбор по типу метаданных;
- режим **Только выбранные**;
- команды **Выбрать все** и **Снять все** для текущего проекта.

Типы метаданных сгруппированы аналогично Навигатору EDT. Подсистемы, общие модули, роли, общие формы,
макеты, картинки, языки, стили, сервисы и другие общие объекты находятся внутри ветки **Общие**.
Верхнеуровневые объекты раскрываются до форм, реквизитов, табличных частей, команд, макетов и других
вложенных объектов. Каждый объект отмечается независимо; отметка групповой папки применяется ко всем
объектам внутри неё. Поиск по имени вложенного объекта сохраняет в результате весь путь к нему.

Поиск начинается со второго введённого символа. Если по имени найдено не более 300 объектов, форма
автоматически раскрывает пути к ним. Содержимое найденного объекта при этом не раскрывается: его
вложенные объекты (совпадающие с запросом только через имя родителя) доступны при ручном раскрытии.
При большем количестве совпадений дерево остаётся свёрнутым, но фильтруется. После очистки строки
поиска дерево сворачивается.

Изменения применяются только после нажатия **ОК**. При закрытии формы с несохранёнными изменениями
плагин попросит подтверждение.

Если ранее добавленный объект был удалён из конфигурации, плагин автоматически удалит его устаревшую
запись из избранного при следующем открытии формы управления. Очистка выполняется только после успешной
загрузки конфигурации и учитывает вложенные объекты.

### Горячие клавиши

**Ctrl+Alt+K** - добавить текущий элемент в избранное или удалить его оттуда (то же, что
«Добавить в избранное/Удалить из избранного»
в меню). Подпись клавиши видна справа в контекстном меню.

**Ctrl+Alt+F** - открыть форму управления избранным. Комбинация работает из любой части окна EDT.

Переназначить: **Window -> Preferences -> General -> Keys**, найти команды «Добавить/удалить из избранного»
и «Избранное (управление)» (категория «Избранное»). После смены комбинации подпись в меню обновится сама.

## Фильтр «Только избранное»

На панели инструментов Навигатора есть кнопка-звезда (переключатель). Нажата - в дереве видно только
избранное, отжата - видно всё как обычно.

Как именно фильтр показывает дерево:

- **Избранный проект без отдельных избранных объектов** виден целиком - со всеми папками и объектами
  («важен проект как таковой»).
- Как только внутри проекта вы добавите в избранное **хотя бы один объект** - проект сворачивается до
  избранных объектов и пути к ним; остальное скрывается. Чтобы снова увидеть проект целиком,
  удалите из избранного все внутренние объекты соответствующей командой на проекте.
- **Проект не из избранного** виден только если внутри есть избранные объекты - и только они.

## Значки избранного в дереве

Избранные проекты и объекты помечаются значком-звездой (декоратор). Если значков не видно,
включите декораторы: **Window -> Preferences -> General -> Appearance -> Label Decorations** -
поставить галку у декоратора избранного.

## Где хранится избранное

Список избранного хранится в служебной папке плагина внутри воркспейса EDT
(`<workspace>/.metadata/.plugins/edt.metadata.favorites/pinned.yaml`). Он **не** попадает ни в один проект
и в его систему контроля версий (git) - это ваши личные настройки рабочего места, у каждого свои.

Элементы избранного привязаны к паре «имя проекта + внутренний идентификатор объекта», поэтому при одновременно
открытых нескольких конфигурациях они не путаются между собой, а переименование объекта или реквизита
избранное не сбрасывается.

## Частые вопросы

**Не вижу пункты меню плагина.** Убедитесь, что вызываете меню именно в панели «Навигатор» 1С:EDT
(команды ограничены ею).

**В меню нет подписи горячей клавиши.** Скорее всего, комбинация конфликтует с другой командой в
том же контексте - Eclipse тогда её отключает. Проверьте Preferences -> Keys по своей комбинации и
при необходимости назначьте свободную.

**Добавил объект в избранное, а проект в фильтре «схлопнулся».** Это ожидаемо: первый отдельный избранный
объект переключает проект из режима «виден весь» в режим «только избранное». Удалите внутренние объекты
из избранного, чтобы вернуть показ проекта целиком.
